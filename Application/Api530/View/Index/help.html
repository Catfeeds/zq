推荐怎么玩?

  • 1、注册成功后，即可免费参与推荐；
  • 2、足球亚盘：周一至周五每天可推荐5场，周六至周日每天可推荐10场
  • 3、足球竞彩：周一至周五每天可推荐5场，周六至周日每天可推荐10场
  • 4、篮球亚盘：周一至周五每天可推荐5场，周六至周日每天可推荐10场

如何获取积分?

  • 1、推荐时系统自动提供100积分进行推荐，不会扣除原账户积分
  • 足球亚盘：赢 = 推荐指数 * 投注积分；赢半 = (推荐指数*积分)/2；
  • 足球竞彩：赢 = (推荐指数-1) * 投注积分；
  • 篮球亚盘：赢 = 推荐指数 * 投注积分；
  • 输、输半、平都不扣积分；
  • 2、玩推荐时填写分析、分享推荐均可获得对应积分
  • 3、坚持每日签到，完成宝箱任务，解锁我的成就都可获得丰富积分

如何获取{$currency}？

  • 1、积分兑换{$currency}；
  • 2、推荐周榜和月榜排名前10名,奖励{$currency}；
  • 3、充值购买{$currency}

积分、{$currency}有什么用？

  • 积分达到5000可兑换{$currency}(1000积分=1{$currency})，{$currency}可购买其他会员发布的推荐内容或提现

如何获取销售提成?

  • 发布推荐内容被其他会员购买后，可获得分成
  • 严禁注册多个ID推荐，若发现作弊等情况，将取消获奖资格；本推荐最终解释权归彩票推荐大师全球体育所有。

等级划分规则和推荐定价

  • 1、足球亚盘胜率 = [ 赢的场次 + 赢半的场次 ] / [ 赢的场次 + 赢半的场次 + 输的场次 + 输半的场次 ] * 100%
  • 2、竞彩胜率 = 赢 / [ 赢 + 输 ] * 100%
  • 3、根据玩法，等级和价格分开计算。
  • 4、升级规律，从推荐第一场赛事算起，一周为一个周期，每达到升级条件则升级成功；
  • 5、降级规律，维持条件则保持等级，不满足条件则降一级，一下一级胜率及场次为标准进行处理；
