推荐怎么玩?

  • 1、注册成功后，即可免费参与推荐；
  • 2、足球亚盘：周一至周四每天可推荐5场，周五至周日每天可推荐10场
  • 3、足球竞彩：周一至周四每天可推荐5场，周五至周日每天可推荐10场
  • 4、篮球亚盘：周一至周四每天可推荐5场，周五至周日每天可推荐10场

如何获取积分?

  • 1、推荐时系统自动提供100积分进行推荐，不会扣除原账户积分
  • 足球亚盘：赢 = 推荐指数X投注积分；赢半 = （推荐指数X积分）/2；
  • 足球竞彩：赢 = （推荐指数-1）X投注积分；
  • 篮球亚盘：赢 = 推荐指数X投注积分；
  • 输、输半、平都不扣积分；
  • 2、玩推荐时填写分析、分享推荐均可获得对应积分
  • 3、坚持每日签到，完成宝箱任务，解锁我的成就都可获得丰富积分

如何获取{$currency}？

  • 1、积分兑换{$currency}；
  • 2、推荐周榜和月榜排名前10名、季榜前5名奖励{$currency}；
  • 3、充值购买{$currency}

积分、{$currency}有什么用？

  • 积分达到5000可兑换{$currency}(1000积分=1{$currency})，{$currency}可购买其他会员发布的推荐内容或提现

怎么是销售分成？

  • 发布推荐内容被其他会员购买后，可获得50%分成
  • 严禁注册多个ID推荐，若发现作弊等情况，将取消获奖资格；本推荐最终解释权归彩票推荐大师全球体育所有。
