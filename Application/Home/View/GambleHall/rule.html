推荐规则_全球体育网
  • 推荐怎么玩?
    1.注册成功后，即可免费参与推荐；
    2.亚盘：周一至周五每天可推荐5场，周六至周日每天可推荐10场
       竞彩：周一至周五每天可推荐5场，周六至周日每天可推荐10场
  • 如何获取积分?
    1.推荐时系统自动提供100积分进行推荐，不会扣除原账户积分
    亚盘：胜 = 推荐赔率 * 推荐积分；

    胜半 = (推荐赔率 * 积分)/2；

    竞彩：胜 = (推荐赔率-1) * 推荐积分；
    2.玩推荐时填写分析和分享推荐结果可获10积分
    3.APP用户坚持每日签到、完成宝箱任务、解锁我的成就均可获得丰厚积分
  • 如何获取Q币？
    1.积分兑换Q币
    2.推荐周榜和月榜排名前10名,奖励Q币
    3.充值购买Q币
  • 积分、Q币有什么用？
    积分达到5000可兑换Q币(1000积分=1Q币)，Q币可购买其他会员
    发布的推荐内容或提现
  • 什么是销售分成？
    1.发布推荐内容被其他会员购买后，可获得分成
    严禁注册多个ID推荐，若发现作弊等情况，将取消获奖资格；
    本推荐最终解释权归全球体育所有。

1、足球亚盘胜率 = [ 赢的场次 + 赢半的场次 ] / [ 赢的场次 + 赢半的场次 + 输的场次 + 输半的场次 ] * 100%

2、竞彩胜率 = 赢 / [ 赢 + 输 ] * 100%

3、根据玩法，等级和价格分开计算。

4、升级规律，从推荐第一场赛事算起，一周为一个周期，每达到升级条件则升级成功；

5、降级规律，维持条件则保持等级，不满足条件则降一级，一下一级胜率及场次为标准进行处理；

  • 周榜分别统计亚盘和竞彩当天起前7日的推荐，按胜率排行，连续3天没有推荐，或参与推荐数不够15场，则会掉出排行榜，重新推荐即可恢复排行。
